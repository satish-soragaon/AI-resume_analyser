# AI Resume Analyzer

Upload your resume and get instant AI-powered feedback — score, ATS tips, skill analysis, and job description matching.

🔗 **Live Demo:** [https://ai-resume-analyser-xyo4.onrender.com](https://ai-resume-analyser-xyo4.onrender.com)

## Tech Stack

- **Backend:** Python, Flask
- **AI:** Google Gemini 2.5 Flash
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render

## Setup

1. Clone the repo and install dependencies
   ```bash
   pip install -r requirements.txt
   ```

2. Set your Gemini API key
   ```bash
   export GEMINI_API_KEY="your_api_key_here"
   ```

3. Run
   ```bash
   python app.py
   ```
